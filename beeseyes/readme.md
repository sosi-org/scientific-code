# Bees eyes

Given coordinates of a bee's eyes and a 2D picture,
calculates input to each eye cell efficiently (by sampling pixels).



## run
run `source ./p3/bin/activate`
run `pycode/pydemo.py`

### Run on MacOS
see `how.sh`
